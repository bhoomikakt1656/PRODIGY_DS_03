# Task-03: Decision Tree Classifier (Bank Marketing)

## Objective
To build a decision tree classifier to predict whether a customer will purchase a banking product using demographic and behavioral data.

## Dataset
Bank Marketing Dataset from UCI Machine Learning Repository.

## Tool Used
- Power BI

## Methodology
- Data cleaning and preprocessing
- Decision tree-based analysis using Key Influencers visual
- Visualization of influential features affecting customer purchase

## Results
Age, duration, previous campaign interactions, and job type significantly influence customer purchase decisions.

## Conclusion
The decision tree model effectively predicts customer behavior and supports targeted marketing strategies.
